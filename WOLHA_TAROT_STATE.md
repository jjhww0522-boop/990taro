# WOLHA TAROT STATE

## 1. 프로젝트 정체성
- 이름: 월하 타로 (Wolha Tarot)
- 테마: 동양 오컬트 다크 판타지, 수묵화, 붉은 달빛, 무속 신앙
- 메인 폰트: 구글 `Gowun Batang` (고운 바탕) - 서늘하고 정갈한 명조체

## 2. 핵심 에셋 및 디자인 규칙
- 배경 이미지: `/cards/main_bg2.jpg` (고정된 전체화면, 흑색 반투명 오버레이)
- 카드 뒷면: `/cards/back_00.jpg`
- 컬러 팔레트:
  - 깊은 흑색 `#121212`
  - 톤다운된 양피지색 `#A39E93`
  - 말라붙은 핏자국/인주색 `#8C2727`
- 대화창(Input):
  - 제미나이 스타일 박스
  - 다크 글래스모피즘(블러 처리)
  - 최대 너비 고정
  - 촌스러운 기본 포커스 테두리 절대 금지

## 3. 구동 흐름 (Phases) 및 상태 로직
- `input`: 사용자가 고민을 입력.
- `deck` / `shuffling`: 절제된 셔플 애니메이션.
- `selecting`:
  - `-ml` 마이너스 마진으로 겹쳐진 카드 가로 스크롤
  - 카드를 뽑을 때 50% 확률로 역방향(`isReversed`) 결정
- `result`:
  - 상단: 정/역방향 라벨(작고 톤다운된 빈티지 스타일, `/` 기호 사용)
  - 중간: 카드 이미지 (역방향일 경우 180도 스핀 및 핏빛 오버레이)
  - 하단: 이름/키워드 출력

## 4. 다음 목표
- 사용자의 질문(고민)과 뽑힌 3장의 카드(정/역방향 포함) 데이터를 모아 AI 모델(LLM) 프롬프트로 전송
- 해석 응답을 받아 화면에 출력하는 기능 구현 예정
