**💡 핵심 개요:** 대화 중 사용자가 추가 질문 시 마이너 카드 1장을 뽑아 AI 답변의 근거로 활용

#### 👨‍💻 Dev팀

| 작업 | 파일 | 설명 |
|------|------|------|
| 마이너 56장 데이터 | `src/lib/minorArcanaData.ts` [NEW] | 4수트(지팡이·잔·검·동전) × 14장, 이름·정방향·역방향 키워드 |
| 채팅 메시지 타입 확장 | `src/app/result/page.tsx` | `ChatMessage`에 `type: "card_pick" \| "card_result"` 추가 |
| 카드 선택 UI 컴포넌트 | `src/components/MinorCardPick.tsx` [NEW] | 대화창 내부에 카드 뒷면 5장 → 1장 선택 UI |
| AI 프롬프트 확장 | `src/app/api/chat/consult/route.ts` | 선택된 마이너 카드 정보를 컨텍스트에 추가 |

#### 🎨 Design팀

- 카드 뒷면 이미지 1종(`back_00.jpg`) 재활용 = Phase 1
- 5장이 대화 버블 폭(약 300px)에 맞게 소형으로 표시 (높이 100px)
- 선택 시 **카드명 + 정/역 키워드** 텍스트로 표시 (이미지는 Phase 2)
- `"✦ 카드를 더 뽑아볼까요?"` 골드 버튼으로 트리거

#### 👔 Ops팀

- 무료 사용자: 추가 카드 뽑기 **1회만** → 이후 결제 유도
- 프리미엄: 무제한
- 56장 개별 이미지는 Phase 2 (나노바나나)

**우선도:** 🟡 중간 | **예상 작업량:** 3~4시간

### 목표 2️⃣ GPT 토큰 악용 방지 시스템

**💡 핵심 개요:** 특정 사용자가 토큰을 과도하게 소비하는 행위를 다층으로 차단

#### 👨‍💻 Dev팀

| 계층 | 구현 | 설명 |
|------|------|------|
| **L1. 요청 횟수 제한** | Upstash Redis | 무료: 해석 1회/일 + 대화 2턴/일, 프리미엄: 해석 무제한 + 대화 5턴/세션 |
| **L2. 입력 길이 제한** | `route.ts` 검증 | 사용자 메시지 최대 500자, 초과 시 자동 잘림 |
| **L3. 속도 제한** | Vercel Edge Middleware [NEW] | IP당 10req/분, 초과 시 429 반환 |
| **L4. 무의미 입력 차단** | `isMeaninglessQuestion()` 강화 | 이미 구현된 함수에 반복 입력 감지 추가 |
| **L5. max_tokens 하드캡** | OpenAI 호출 | 해석: 2000, 대화: 900 (현재 유지) |

```
파일 변경 목록:
- src/middleware.ts [NEW] → Edge rate limiter
- src/app/api/chat/route.ts → 입력 길이 검증 추가
- src/app/api/chat/consult/route.ts → 입력 길이 검증 추가
```

#### 👔 Ops팀

- Redis 없이도 L2·L4·L5는 즉시 적용 가능 (코드 변경만)
- Redis 연결 후 L1·L3 활성화
- **비용 모니터링:** OpenAI 대시보드에서 일일 지출 한도($5) 설정 권고

**우선도:** 🔴 긴급 | **예상 작업량:** 2시간

### 목표 3️⃣ GPT 프롬프트 토큰 최적화

**💡 핵심 개요:** 토큰을 아끼면서도 사용자가 만족하는 성의 있고 긴 답변 유지

#### 👨‍💻 Dev팀

| 최적화 | 현재 | 변경 | 절감률 |
|--------|------|------|--------|
| **모델** | `gpt-4o-mini` | 유지 (비용 최적) | — |
| **시스템 프롬프트 압축** | ~800자 | ~400자로 핵심만 (중복 제거) | ~50% |
| **대화 히스토리 윈도우** | 전체 전송 | **최근 6턴만** 전송 | ~40% |
| **초기 해석 요약** | 5필드 전문 전달 | 핵심 키워드만 요약(100자 이내) | ~60% |
| **응답 길이 지시** | "400자 이상" | "350~450자" 범위 지정 + `max_tokens: 700` | 안정화 |

```
파일 변경 목록:
- src/app/api/chat/route.ts → 시스템 프롬프트 압축
- src/app/api/chat/consult/route.ts → 히스토리 윈도우 + 요약
```

#### 👔 Ops팀

- 현재 gpt-4o-mini: $0.15/1M input, $0.60/1M output
- 최적화 후 1회 해석 비용: ~$0.003 → ~$0.0015 (50% 절감)
- 1회 대화 비용: ~$0.002 → ~$0.001

**우선도:** 🔴 긴급 | **예상 작업량:** 1.5시간
